# 빅데이터 분석 특강

* 주제 : 공공데이터 활용 빅데이터 분석 with 파이썬
  - 내가 관심있는 또는 해결하고 싶은 문제를 정의하고,
  - 데이터 기반으로 문제를 해결한다.

* 과정
  1. 데이터분석의 이해
  2. 파이썬 기초 - 변수, 반복문, 조건문, 외부라이브러리 가져오기 등
  3. 데이터 분석을 위한 파이썬 기초 - pandas, numpy, matplotlib 등
  4. 데이터 수집 - 공공데이터 수집 및 전처리
  5. 데이터 분석 - 모델링 및 시각화
  6. 개인 프로젝트 설계 및 발표
